# 🤖 Task 4 - Context-Aware Chatbot Using RAG

## 📌 Objective

The objective of this project is to build a Retrieval-Augmented Generation (RAG) chatbot that answers user queries using a custom knowledge base. The chatbot retrieves the most relevant information using vector similarity search and presents context-aware responses through an interactive Gradio interface.

---

## 📂 Knowledge Base

A custom AI & Machine Learning knowledge base was created containing topics such as:

- Artificial Intelligence
- Machine Learning
- Deep Learning
- Natural Language Processing
- Computer Vision
- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning
- BERT
- RAG
- LangChain
- FAISS
- Embeddings
- Vector Databases

---

## 🚀 Methodology

1. Created a custom knowledge base.
2. Converted text into LangChain Documents.
3. Split documents into smaller chunks.
4. Generated embeddings using Sentence Transformers.
5. Stored embeddings in a FAISS vector database.
6. Configured a retriever for semantic search.
7. Built a chatbot function to retrieve relevant information.
8. Developed an interactive Gradio interface for user interaction.

---

## ✨ Features

- Context-aware question answering
- Semantic search using FAISS
- LangChain document processing
- Sentence Transformer embeddings
- Interactive Gradio web interface
- Retrieval-Augmented workflow
